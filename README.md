# S2T Project

## Project Structure

- `/discovery/` - Discovery and research documentation
- `/truth/` - Truth and facts documentation
- `/requirements/` - Project requirements
- `/architecture/` - System architecture documentation
- `/design/` - Design documents and mockups
- `/mcp/` - MCP related files
- `/backend/` - Backend application code
- `/frontend/` - Mobile app frontend code
- `/tests/` - Test files and test suites
- `/security/` - Security documentation and configurations
- `/audit/` - Audit logs and reports
- `/dx/` - Developer experience tools and documentation
- `/management/` - Project management files
- `/governance/` - Governance and compliance documentation
